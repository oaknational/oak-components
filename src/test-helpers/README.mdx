# test-helpers

Includes conveniences to ease in testing with Oak Components.

### `installMockResizeObserver`

Installs a mock [`ResizeObserver`](https://developer.mozilla.org/en-US/docs/Web/API/ResizeObserver) that can be used to test components that use `ResizeObserver` in an environment where it isn't present.

### `installMockIntersectionObserver`

Installs a mock [`IntersectionObserver`](https://developer.mozilla.org/en-US/docs/Web/API/IntersectionObserver) that can be used to test components that use `IntersectionObserver` in an environment where it isn't present.
